# 1. Basic Environment Setup
## 1.1 Docker Installation
If you already have Docker installed, please skip Section 1.1.

The following installation script is provided by 1ms (Millisecond Mirror).<a href="https://1ms.run/docs/1ms.run/" target="_blank" rel="noopener"> For more details,refer to the official documentation</a>:
```shell
# Obtain root privileges
sudo -i

# Run the installation script
bash <(curl -f -s --connect-timeout 10 --retry 3 https://linuxmirrors.cn/docker.sh) \
  --source mirrors.tencent.com/docker-ce --source-registry docker.1ms.run --protocol https \
  --install-latested true --close-firewall false --ignore-backup-tips
```
Running this script will automatically install Docker with optimized configurations.

## 1.2 Install PostgreSQL and Redis Databases for Basic Runtime

### 1.2.1 aunch Base Services
```shell
# Run base postgres 17.5 image user postgres password 123456
docker run -d --restart=always --name base-db -p 5432:5432 -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=123456 -e POSTGRES_DB=plus_one_base  docker.1ms.run/library/postgres:17.5
# Run base redis 7.2 image password 123456
docker run -d --restart=always --name base-redis -p 6379:6379 -e REDIS_PASSWORD=123456  docker.1ms.run/redis:7.2 redis-server --requirepass 123456
```

### 1.2.2 Clone Source Code and Retrieve Initialization Script

```shell
# Clone the backend repository
git clone https://gitee.com/lkShi/plus-one-system.git
# The database initialization SQL script is located at: /plus-one-system/src/main/resources/db-file/init_db.sql
```
The database initialization SQL script is located at: /plus-one-system/src/main/resources/db-file/init_db.sql

### Execute init_db.sql in the PostgreSQL Container from Step 1.2.1

Connect to the running PostgreSQL container (base-db) and run the init_db.sql script to initialize the database schema and seed data.

# 3. Running the Plus-One UI Frontend

## 3.1 Clone the Frontend Code (Skip if Already Done)

```shell
# Clone the Frontend Code (Skip if Already Done)
git clone https://gitee.com/lkShi/plus-one-ui.git
```

## 3.2 Install nvm (Node Version Manager)

Skip this step if you already have Node.js installed.

Refer to the official nvm installation guide:<a href="https://nvm.uihtm.com/doc/install.html" target="_blank" rel="noopener">ovm Official Documentation</a>

On Linux or macOS, it's recommended to install nvm via Homebrew:
```shell
brew install nvm
```

### 3.2.1 Install Node.js v20

Run the following commands to install and verify Node.js version 20:

```shell
nvm install 20
nvm list
```
## 3.3 Run the Frontend Application

Navigate to the plus-one-ui project directory and execute the following:

```shell
# Install Yarn globally (if not already installed)
npm install -g yarn
```

Then install dependencies and start the development server:

```shell
# Install project dependencies
yarn
# Start local development server
yarn dev
```
