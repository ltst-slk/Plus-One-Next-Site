# 1. 基础环境安装
## 1.1 Docker 安装
如果您已安装 Docker,请跳过步骤 1.1

以下安装脚本为 1ms 毫秒镜像脚本 <a href="https://1ms.run/docs/1ms.run/" target="_blank" rel="noopener">毫秒镜像 - 安装 Docker 文档</a>：

```shell
# 获取权限
sudo -i

# 执行脚本
bash <(curl -f -s --connect-timeout 10 --retry 3 https://linuxmirrors.cn/docker.sh) \
  --source mirrors.tencent.com/docker-ce --source-registry docker.1ms.run --protocol https \
  --install-latested true --close-firewall false --ignore-backup-tips
```
运行脚本即可一键安装

## 1.2 安装基础运行的 PG 以及 Redis 数据库

### 1.2.1 运行基础环境
```shell
# 运行 基础的  postgres 17.5 镜像 用户 postgres 密码 123456
docker run -d --restart=always --name base-db -p 5432:5432 -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=123456 -e POSTGRES_DB=plus_one_base  docker.1ms.run/library/postgres:17.5
# 运行 基础的 redis 7.2 镜像 密码 123456
docker run -d --restart=always --name base-redis -p 6379:6379 -e REDIS_PASSWORD=123456  docker.1ms.run/redis:7.2 redis-server --requirepass 123456
```

### 1.2.2 克隆代码并获取初始化脚本

```shell

git clone https://gitee.com/lkShi/plus-one-system.git
#脚本位于 /plus-one-system/src/main/resources/db-file/init_db.sql
```
脚本位于 /plus-one-system/src/main/resources/db-file/init_db.sql

# 3. Plus-One UI 前端运行
## 3.1 克隆代码
```shell
# 克隆代码 如前置步骤已克隆可忽略
git clone https://gitee.com/lkShi/plus-one-ui.git
```
## 3.2 安装 nvm
如已安装 node 请忽略 3.1
请参考<a href="https://nvm.uihtm.com/doc/install.html" target="_blank" rel="noopener">nvm官网</a>
如果使用 Linux 以及 macos 建议通过 brew 直接安装
### 3.2.1 安装 node 20
执行以下脚本 安装 node 20
```shell
nvm install 20
nvm list
```
## 3.3 运行前端代码
在 plus-one-ui 项目目录 打开控制台 并安装 yarn
```shell
npm install -g yarn
```

通过以下脚本安装依赖并运行
```shell
# 依赖安装
yarn
# 本地运行
yarn dev
```
